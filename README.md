# Cliente Mídia™ — Portal

Centro de autoridade da categoria Cliente Mídia™ no Brasil. Built with Next.js 14, Sanity CMS, Supabase, Resend, and Cloudinary.

## Stack

- **Framework:** Next.js 14 (App Router) + TypeScript
- **Styling:** Tailwind CSS with custom design tokens
- **CMS:** Sanity (editorial content)
- **Database:** Supabase (leads, analytics)
- **Email:** Resend (lead magnet delivery)
- **Images:** Cloudinary (CDN + optimization)
- **Hosting:** Vercel

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Editorial pillars

The portal is organized around 4 content pillars:

1. **Diagnóstico** — the problem of rented reach, the real cost of CAC
2. **Categoria** — defining Cliente Mídia™, distinguishing from UGC/influencer
3. **Pesquisa** — primary data, benchmarks, research reports
4. **Movimento** — the future, interviews, predictions
