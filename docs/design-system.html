A gramática visual da nova categoria.

Este documento define o vocabulário visual do portal Cliente Mídia™. Tipografia, paleta, componentes e princípios editoriais que governam cada artigo, cada página, cada peça de comunicação. Luxo na superfície. Clareza no parágrafo. Autoridade na ausência de barulho.

i.

A marca Cliente Mídia™ se manifesta em três formas: o logotipo principal, o monograma e o marcador editorial. Cada uma serve uma função específica de hierarquia e contexto.

v.

Direção fotográfica.

Toda imagem que aparece no portal segue três princípios não negociáveis: tem peso cinematográfico, tem baixa saturação e respeita a paleta de pretos, cremes e ouros. Foto de banco com cara de banco está banida. Imagem gerada por IA precisa parecer fotografia editorial — não ilustração.

Cinemático · luz baixa
Editorial · provador
Produto · stillife dourado
Luz

Sempre direcional, lateral, com áreas de sombra profunda. Nunca flat. Nunca chapado. Sensação de "câmera grande" e filme analógico.

Cor

Dessaturação calibrada — entre 70% e 85% da saturação original. Tons quentes preservados. Azuis empurrados para o cinza. Sem cores fluorescentes.

Composição

Respiro generoso. Sujeito raramente no centro. Pontos de fuga horizontais. Linhas arquitetônicas explícitas. Ar editorial.

vi.

Princípios editoriais.

Quando houver dúvida sobre uma decisão de design ou de conteúdo, volte a estes seis princípios. Eles são a constituição do projeto.

01.

Luxo, mas acessível.

A embalagem é de revista premium. A linguagem é de conversa entre profissionais. Lojista classe A entende e se reconhece. Lojista de outra classe entende e aprende. Ninguém precisa de tradutor.

02.

Autoridade pela restrição.

Quem precisa gritar não tem autoridade. Cada espaço em branco, cada tipografia contida, cada cor não usada — tudo isso comunica solidez. O barulho fica para a concorrência.

03.

Dados como argumento.

Números reais, com fonte citada, no corpo da prosa. Não como decoração. Não como infográfico vazio. O dado existe para sustentar uma afirmação — nunca para enfeitar.

04.

Frase curta. Ritmo claro.

Período longo só quando justificado. Parágrafo de quatro linhas, no máximo. Frase isolada vale uma seção. O ritmo do texto é parte do design — tanto quanto a tipografia.

05.

A categoria primeiro.

Em qualquer artigo, qualquer relatório, qualquer comunicação: a categoria Cliente Mídia™ é o sujeito principal. A Viralize Luxo aparece como infraestrutura — nunca como protagonista. Centro de autoridade, não folder de venda.

06.

Nada brilhante.

Sem gradientes saturados. Sem botões que pulsam. Sem ouro reluzente. O luxo verdadeiro é fosco, contido, denso. Se um elemento parece "design de site moderno", está errado. Volta para o forno.
